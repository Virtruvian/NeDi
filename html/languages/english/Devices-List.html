Devices List

Introduction

List devices, displaying database properties and even graphs for quick CPU, memory and temperature comparison.

Usage

Simply click Show to list all devices or define the criteria as desired. There are 3 modes of filtering:
  1. 1 criteria matching a column within Condition A (leaving Operator at A only).
  2. 2 criterias like above, but logically combined with Operator.
  3. Alternatively you can compare 2 columns directly by using a combination with "col" (To find devices where firstseen equals lastseen for example).
  4. The selected devices can be added to the monitoring if you click the Monitor button. Go to Monitor-Setup to further configure them.
Click Monitor to add the devices to the Monitoring table. This will allow service checks and event classification.

Details

  • Knowledge about regexp is very helpful when working with this module!
  • You can save searches in Bookmarks.
  • Icon, name, IF and IP address provide quick shortcuts to Device-Status, Nodes-List and the device's CLI.
  • Unselecting the device column hides the icons to simply copy and paste the list as text.
  • The resulting SQL query can be shown using the cog icon, if logged in as admin. It links to Other-Export to generate CSV files or the likes.

Nodes List

Introduction

List nodes, displaying database properties, graphs and available services.

Important!

If you list available services, make sure you don't match too many nodes as it will take a long time to scan the open ports.

Conditions involving criterias (e.g. location or contact) from the devices or interfaces (e.g. IF alias) table, cannot be used to delete nodes due to query restrictions!

Usage

List device Vlans by pressing the "Show" button. Use the "Display" select box to add or remove the columns you wish to see (Ctrl + Mouseclick). The following filter methods are available to narrow down your search:
  1. 1 criteria matching a column within Condition A (leaving Operator at A only).
  2. 2 criterias like above, but logically combined with Operator.
  3. Alternatively you can compare 2 columns directly by using a combination with "col" (To find nodes where firstseen equals lastseen for example).

Details

  • Knowledge about regexp is very helpful when working with this module!
  • You can save searches in Bookmarks.
  • The resulting SQL query can be shown using the cog icon, if logged in as admin. It links to Other-Export to generate CSV files or the likes.
  • Clicking on the NIC vendor icon takes you to Nodes-Status where you get all node details at a glance.

Combination Reports

Introduction

This module allows you to quickly generate reports based on a focus rather than the underlying data. The asset report for example retrieves data from devices and modules tables.

Usage

  • Filter on device based criterion, if desired (won't be taken into account for links to other modules).
  • Select report you wish to create.
  • Select graph(s) and their size you wish to include (if any).
  • Optionally include last map.
  • Increase the limit to see more results.
  • Check 'Alternative Sort' to sort keys rather than values or revers times.

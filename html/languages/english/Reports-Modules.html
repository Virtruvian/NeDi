Module Reports

Introduction

Need to know how many modules of a kind you've got? This report also helps, if you need to generate a HW inventory for support contracts based on serial numbers etc.

Usage

  • Filter on device based criterion, if desired (won't be taken into account for links to other modules).
  • The 'Summary' presents an overview of which modules are installed in which devices.
  • The 'Inventory' lets you generate a detailed list of modules where you can use the filter (on device type) to narrow down the selection.
  • Check 'Alternative' to sort by model (in Summary) or device type (in Inventory) rather than values or device name.
  • Print supplies and virtual machines are specific reports to reflect these items, which are stored in the DB as modules.

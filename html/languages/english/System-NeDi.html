Run NeDi

Introduction

Execute the actual nedi.pl from the GUI.

Usage

The module is divided in to the following parts:
  1. The default is --help, which will options and output legend.
  2. .def list shows all devices definitions available.
  3. Discover will actually query devices.
  4. Init is the most dangerous one here and will only proceed with the right DB admin credentials and after confirmation.

Details

  • It's not intended for a full blown discovery, as on most servers PHP cannot run for more than 30 seconds.
  • Don't worry, if nothing happens right away, give it some time to complete.

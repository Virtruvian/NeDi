Network List

Introduction

List networks by VRFs for example.

Usage

List device networks by pressing the "Show" button. Use the "Display" select box to add or remove the columns you wish to see (Ctrl + Mouseclick). The following filter methods are available to narrow down your search:
  1. 1 criteria matching a column within Condition A (leaving Operator at A only).
  2. 2 criterias like above, but logically combined with Operator.
  3. Alternatively you can compare 2 columns directly by using a combination operatore with "col" (To find modules where type equals description for example).

Details

  • Knowledge about regexp is very helpful when working with this module!
  • You can save searches in Bookmarks.
  • The resulting SQL query is always shown at the end of the table. It can be used in Other-Export to generate CSV files or the likes.
  • The device name links to its status.
